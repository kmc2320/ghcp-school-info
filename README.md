# 神山まるごと高専 紹介ページ

GitHub Pages で公開する、神山まるごと高専のシングルページ紹介サイトです。

- 公開URL: https://kmc2320.github.io/ghcp-school-info
- 技術構成: HTML / CSS / JavaScript（フレームワークなし）

## ローカル確認

`index.html` をブラウザで開くか、VS Code の Live Server で確認できます。

## 画像の追加方法

画像素材は `images/` 配下に配置してください。

例:

- `images/hero.jpg`
- `images/dorm.jpg`

現在は画像未使用でも見た目が崩れないよう、グラデーション背景を使っています。画像を使う場合は `css/style.css` の該当セクションに `background-image` を設定してください。
